# 🔐 Cyber Password Intelligence Analyzer

An advanced password security analyzer built using HTML, CSS, and JavaScript that evaluates password strength, calculates entropy, estimates crack time, and provides security recommendations.

## 🚀 Features

## Features

- Password Strength Analysis
- Entropy Calculation
- Crack Time Estimation
- SHA-256 Hash Generation
- RTX 4090 Attack Simulation
- Attack Resistance Matrix
- NIST Compliance Check
- Breach Detection
- Enterprise Security Policies
- Smart Password Generator
- JSON Security Report Export
- Personal Information Detection
- Keyboard Pattern Detection

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- DOM Manipulation
- Regular Expressions (Regex)
